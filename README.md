# flutter_github_actions

This project is flutter uses GitHub actions sample.

## Env

Repository Secrets

|Key|Value|
|---|---|
|PERSONAL_ACCESS_TOKEN_GITHUB|base64(username:token)|
|MATCH_PASSWORD|fastlane match password|
|FIREBASE_TOKEN|`firebase login:ci`|
